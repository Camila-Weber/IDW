A avaliação 1 consiste na criação de um web site em HTML utilizando os conceitos estudados na Unidade 01.
Os requisitos são:

  1. [2.0pt] Deve conter um menu com no mímino 5 links, cada link deve abrir uma página html, todas as páginas devem conter link para todas as outras páginas;
  2. [2.0pt] A segunda página deve conter uma lista não ordenada com uma sublista ordenada em caracteres alpha (A. B. C. D) em maiúsculo;
  3. [2.0pt] A terceira página deve conter 2 tabelas, uma com coluna mesclada, outra com linha mesclada;
  4. [2.0pt] A quarta página deve conter um 1 formulário com os seguintes campos:
    1. Texto simples
    2. Texto de múltiplas linhas
    3. Caixa de seleção com opções pré-definidas
    4. Botões de rádio e de múltipla escolha
    5. Botão de envio de arquivo e botão para envio do formulário.
  5. [2.0pt] A quinta página deve conter um vídeo do youtube em um quadro interno e um áudio (arquivo mp3);

Obs: O uso de CSS é opcional [1.0pt extra]
